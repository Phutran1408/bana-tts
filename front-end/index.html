Typing Guide
Special Characters:
u8, uu, u\ → ŭ
c6, cc, c\ → ĉ
e8, ew, e\ → ĕ
i8, iw, i\ → ĭ
n4, nx, n\ → ñ
o8, o\ → ǒ
aw, a8 → ă
aa, a6 → â
dd, d9 → đ
ee, e6 → ê
oo, o6 → ô
ow, o7 → ơ
uw, u7 → ư
Tips:
• Case doesn't matter for 2nd character (Uu → Ŭ)
• Repeat same character to cancel (e66 → e6)
• Type naturally - replacement happens instantly
• Turn off Unikey on laptop/PC or switch to English keyboard on phone
